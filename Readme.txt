# Spice Haven - Restaurant Website

Spice Haven is a fully responsive and modern restaurant website designed to showcase the restaurant's menu, ambiance, and services. 
I built this website using "Bootstrap" and customized it extensively to create a unique and user-friendly experience.


# Features
- Responsive Design*: Optimized for all devices (desktop, tablet, and mobile).
- Custom Styling*: Modified the design to align with the restaurant's brand and theme.
- Interactive Elements*: Added animations, a dynamic menu section, and a reservation form.
- Gallery*: A visually appealing section to showcase the restaurant's ambiance and dishes.

# Technologies Used
- HTML5, CSS3, Bootstrap 5, JavaScript

# How to Use
1. Download or clone the project files.
2. Open `index.html` in your browser to view the website.
